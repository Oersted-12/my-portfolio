The Vision

This group project focused on analyzing vehicle-related systems using technology-driven approaches. It highlighted system design, problem-solving, and the application of engineering and software concepts to real-world transportation scenarios.

  • Secure RBAC: Role-Based Access Control for Customers, Staff, and Administrators.
  • Digital Key Integration: Concept interface for secure vehicle access and monitoring.
  • Dynamic Booking: Real-time availability management and automated rental scheduling.
  • Admin Intelligence: Comprehensive analytics dashboard for fleet management.
